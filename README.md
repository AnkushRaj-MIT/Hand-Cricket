# Hand Cricket Game
A browser-based Hand Cricket game built using HTML, CSS, and JavaScript. Challenge the computer, win the toss, bat or bowl, score runs, and chase targets in this digital version of the classic hand cricket game.
## Features
- Toss system
- User vs Computer gameplay
- Batting and Bowling modes
- Live score tracking
- Target chasing
- Win/Loss/Draw results
- Play Again option
- Responsive and interactive UI
## Technologies Used
- HTML5
- CSS3
- JavaScript
## How to Play
1. Start the game and play the toss.
2. Choose to bat or bowl if you win the toss.
3. Select a number on each turn.
4. If your number matches the computer's number, you're out.
5. Score more runs than your opponent to win.
